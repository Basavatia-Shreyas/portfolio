## Development Tools
In order to complete this project, I used the following tools
- Next.js
- React.js
- Google Cloud Vision
- Firebase


## Features
With FotoMind, users can login and securley upload their own photos to the platform. 
![](https://res.cloudinary.com/dmpauhme6/image/upload/v1717692520/cwxqacklf071w1l5yvff.png)

FotoMind automatically uses Google Cloud Vision to run computer vision models that label images with their appropriate tags. Users can then search for people, places, or things in their photos an FotoMind will display relevant photos 
![](https://res.cloudinary.com/dmpauhme6/image/upload/v1717692526/griwbnnjio3ofg4n1lhf.png)
