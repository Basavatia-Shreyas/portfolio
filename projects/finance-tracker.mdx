## Overview
The finance tracker is a software as a service platform that allows users to track their spending habits over time to make smarter financial decisions. The home page consists of cards that clearly show expenses, income, and remaining money. It also shows an interactive graph to plot expenses and income as well as a pie chart to show expenses accross spending categories. 
## Development Stack
In order to complete this project, I utilzed the following tech stack
- ProstgresSQL
- Drizzle ORM
- React.js
- Next.js
- Tailwind CSS
- Hono.js
- Clerk Authentication

## Overview Page
![](https://res.cloudinary.com/dmpauhme6/image/upload/v1717123673/sh5cmgwi2ommgvenmuyr.png)
- View income, expenses, and remaining money by filtering by account and date rate
- View key insights into your expenses and income using the area chart which can be swapped for a line graph or bar chart
- Track expenses by category using the pie chart which can be swapped for a radial chart or target chart

## Transactions Page
![](https://res.cloudinary.com/dmpauhme6/image/upload/v1717123976/tz8xofzhgvi2zg81aqh6.png)
- Detailed list of transactions incdicating the category, payee, account, amount, and date of each transaction
