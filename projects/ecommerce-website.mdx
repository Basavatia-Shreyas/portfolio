## Development Stack
In order to complete this project, I rapidly gained an understanding in the following development stack
- MySQL
- Prisma
- React.js
- Next.js
- Tailwind CSS

## Features of the Admin Dashboard
![](https://res.cloudinary.com/dmpauhme6/image/upload/v1698249794/ejwou5dwt5h0l6wp0pcu.jpg)
- Easily create and manage stores, products, categories, and billboards
- Track orders, sales, and revenue with graphs for key insights
- Secure Clerk authentication for users to login

## Features of the Store
![](https://res.cloudinary.com/dmpauhme6/image/upload/v1698252365/ort3iwtdhcs6vm4itcq2.png)
- Detailed category and product pages
- Search and filter by category
- Product quick view
- Fully functional Stripe payments
