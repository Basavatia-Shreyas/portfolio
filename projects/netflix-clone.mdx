## Development Stack
In order to complete this project, I utilized the following development stack
- MongoDB
- Prisma
- React.js
- Next.js
- Tailwind CSS

## Features of the Netflix clone
![](https://res.cloudinary.com/dmpauhme6/image/upload/v1702029614/eteih0su82eyuqqvls8j.png)

- Secure authentication to create your own account
- Fully functional video player
- Dashboard page to browse trending films
- Sort films by series, movies, new & popular, language, and more
- Info-Modal to get quick information about a film
- My List feature for your favorite films
- Fluent animations just like Netflix
